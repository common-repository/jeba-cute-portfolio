Jeba Cute portfolio is an awesome portfolio, super lightweight plugin for your wordpress website to show portfolio. 

== Description ==

This plugin will enable awesome portfolio in your wordpress theme. You can use the jeba cute portfolio via shortcode in everywhere you want, even in theme files. 

Wanna see how it works? Click here: http://prowpexpert.com/jeba-cute-portfolio/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want use the jeba cute portfolio in your theme php, Place `<?php echo do_shortcode('[jeba_portfolio]'); ?>` in your templates.
5. In the [jeba_portfolio] shortcode you can change post_type="", category="", count="" . Here count is total post. 
Shortcodes

<strong>Use Jeba Cute portfolio shortcode</strong>
<pre>[jeba_portfolio]</pre>

<strong>Here can change</strong>
<pre>[jeba_portfolio post_type="" category="" count=""]</pre>

== Frequently Asked Questions ==
= How can change post_type="" category="" count=""? =
In "post_type" you can use "post", "page" and other "custom_post_type". In "category" you can use your post category to show category based. In "count" can use total how many post want to be show.
